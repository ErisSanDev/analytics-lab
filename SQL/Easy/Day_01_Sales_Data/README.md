<h1 align="center">Day 01 - SQL Practice</h1>

This project contains basic SQL practice questions using a simple sales database.

The database consists of three tables:

* **Customers** – stores customer information such as customer ID, name, and city.
* **Products** – stores product details including category and price.
* **Orders** – stores customer purchase records.

## What I Practiced

### Question 1

Retrieved all products that belong to the Furniture category.

### Question 2

Identified customers who have placed more than one order by using aggregation and filtering.

### Question 3

Combined customer and order data to display each order along with the customer name.

### Question 4

Displayed all customers, including those who have never placed an order, using a LEFT JOIN.

### Question 5

Calculated the total quantity sold for each product and ranked products based on sales.

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* HAVING
* COUNT
* SUM
* INNER JOIN
* LEFT JOIN
* ORDER BY
* Aggregate Functions

## Learning Outcome

Through these exercises, I learned how to filter data, summarize records, join multiple tables, and create business-focused reports using SQL.
